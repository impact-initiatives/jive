# Local
Just used for local testing and development of some components.

Some of these components may be out of date.

## Local Development & Configuration

### Prerequisites
* Python 3.12+
* [uv](https://docs.astral.sh/uv/) package manager

### Project Environment Setup
To create the python environment run:
```bash
uv sync  --all-extras
```

### Environment Variables
Environment variables are set manually in each of the files as required.
